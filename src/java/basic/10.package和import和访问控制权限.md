# package和import
包机制的作用是为了方便程序的管理。不同功能的类分别放在不同的包下

package是一个关键字，后面加包名。例如

```
package com.clubfactory.redis
```

注意：package语句只允许出现在java源代码的第一行

包名的命名规范一般是公司域名倒序的方式
